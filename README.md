# Insane Black Hole Portfolio

A 3D immersive portfolio featuring a stylized black hole with accretion disk, gravitational lensing effects, and interactive camera controls.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Features

-   **React Three Fiber** for 3D rendering.
-   **Custom Shaders** for the accretion disk.
-   **Post-processing** (Bloom) for the neon glow effect.
-   **Camera Rig** for parallax mouse interaction.
-   **Glassmorphism UI** for a modern feel.
